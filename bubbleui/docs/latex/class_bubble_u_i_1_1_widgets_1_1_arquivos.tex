\doxysubsection{Detailed Description}
\doxylink{class_bubble_u_i_1_1_widget}{Widget} para exibir e interagir com arquivos e pastas em um diretório. 

Esta classe permite renderizar uma interface gráfica para exibir e manipular arquivos e pastas contidos em um diretório específico. Contém elementos interativos, como botões e uma caixa de texto.

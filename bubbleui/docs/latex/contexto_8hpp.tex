\doxysubsection{Detailed Description}
Declara a estrutura {\ttfamily Contexto} e funções relacionadas. 

\begin{DoxyCopyright}{Copyright}

\end{DoxyCopyright}


A estrutura {\ttfamily Contexto} serve como base para armazenar informações gerais sobre a aplicação, incluindo a janela GLFW, inputs e painéis associados. Também gerencia a renderização e a atualização de eventos no framework \doxylink{namespace_bubble_u_i}{Bubble\+UI}.
